Ignition.js & AudioTools.js Documentation

JavaScript library for ML operations and browser audio features

Getting Started

Include the libraries in your project:

<script src="ignition.js"></script>
<script src="audiotools.js"></script>

Ignition.js Functions

topk(arr, k)

Returns the top k largest numbers from an array.

topk([3,1,4,2,4], 2); // [4,4]

plus(a, b)

Adds two numbers.

plus(5, 3); // 8

tensor(...numbers)

Multiplies up to 5 numbers.

tensor(2,3,4); // 24 (simple tensor)

securetensor(...args)

Multiplication in a sandbox for security.

securetensor(2,3,5);

params(temp)

Sets a global x if undefined.

params(42);

train(file)

Reads file content using FileReader.

train(myFile);

AudioTools.js Functions

newSpeech()

Creates a new SpeechRecognition instance.

newSpeech();

startSpeech()

Starts speech recognition.

startSpeech();

toEnglish()

Sets recognition language to English (US).

toEnglish();

talkNow(text)

Converts text to speech using SpeechSynthesis.

talkNow("Hello world");

audparams(temp)

Sets a global variable y if undefined.

audparams(0.7);

Example Usage

// ML example
const arr = [10, 20, 5, 30];
console.log(topk(arr, 2)); // [30, 20]

// Audio example
newSpeech();
toEnglish();
talkNow("Testing audio synthesis.");

// Secure multiplication
securetensor(2, 3, 5);

// File reading
const fileInput = document.createElement('input');
fileInput.type = 'file';
fileInput.addEventListener('change', (e) => {
  train(e.target.files[0]);
});
document.body.appendChild(fileInput);

Notes

  • Use securetensor() for sensitive computations.
  • Speech recognition requires browser permission.
  • FileReader logs file content in the console.
