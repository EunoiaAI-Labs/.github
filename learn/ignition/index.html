Ignition.js Documentation

JavaScript library for basic ML operations and secure tensor computation

Getting Started

Include ignition.js in your project:

<script src="ignition.js"></script>

Make sure to include any dependent files like audtools.js if needed.

Functions

1. topk(arr, k)

Returns the top k largest numbers from an array.

const arr = [3, 1, 4, 2, 3, 4, 4];
console.log(topk(arr, 3)); // Output: [4, 4, 4]

2. plus(equat, equat1)

Adds two numbers together.

console.log(plus(5, 3)); // Output: 8

3. tensor(logic, logic1, logic2, logic3, logic4)

Multiplies up to 5 numbers together. Only works with defined numbers.

tensor(2, 3, 4); // Multiplies 2 * 3 * 4
// Note: This is a simple tensor, for secure operations use securetensor()

4. securetensor(...args)

Performs multiplication in a sandboxed environment for security.

securetensor(2, 3, 4); // Securely multiplies 2 * 3 * 4 in sandbox

5. params(temp)

Sets a global variable x if undefined.

params(42); // Sets x = 42 if x is undefined

6. train(file)

Reads a file and logs its contents (can be used to train models).

const fileInput = document.getElementById('fileInput');
fileInput.addEventListener('change', (e) => {
  train(e.target.files[0]);
});

Example Usage

Basic example combining functions:

const arr = [10, 20, 5, 30];
console.log("Top 2:", topk(arr, 2));

const product = securetensor(2, 3, 5);
console.log("Secure product computed!");

// Training a file
const fileInput = document.createElement('input');
fileInput.type = 'file';
fileInput.addEventListener('change', (e) => {
  train(e.target.files[0]);
});
document.body.appendChild(fileInput);

Notes

  • tensor() is not secure; prefer securetensor() for sensitive operations.
  • Functions like params() set global variables only if undefined.
  • File reading via train() uses FileReader and logs file content to the console.
